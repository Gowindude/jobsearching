## 🔄 How to update this list

Open Claude Code in this repo directory and paste the following prompt. The efficient approach uses Bash to curl Greenhouse/Lever APIs directly (zero LLM tokens for API scraping), then a single agent for web searches.

```
Search for open aerospace and defense internships for Summer 2027 and Spring 2027.

**Filters — apply to every role:**
- US-based only: exclude any role where the primary work location is outside the United States. For Palantir "France" / "Poland" cohorts, exclude even if the listed city is US-based, since the deployment is abroad.
- Technical roles only: include software engineering, hardware engineering, electrical engineering, mechanical engineering, GNC, avionics, propulsion, flight test, systems engineering, embedded systems, autonomy, aerodynamics, simulation, CFD, FEA, manufacturing engineering, perception, robotics, FPGA, forward deployed software engineer. Exclude: business development, operations, people ops, accounting, marketing, communications, supply chain, finance, HR, legal, procurement, construction management, field marketing, community engagement.

**Step 1 — Bash API scrape (run in terminal, no LLM needed):**

```bash
SCRATCHPAD="/tmp/intern_scrape"
mkdir -p "$SCRATCHPAD"

GH_SLUGS=(spacex rocketlab relativityspace stokespace firefly ablspacesystems
  vast vastspa jobyaviation archeraviation wiskaero betatechnologies supernal
  andurilindustries shieldai kratosd leidos saic hermeus ursamajor
  firehawkaerospace venusaerospace sarcos xwing flyzipline trueanomalyinc
  physicsxai nearspacelabs luminarycloud dedrone fortemtechnologies
  aerojetrocketdyne apexspace vardaspace starfishspace turionspace)

echo "=== GREENHOUSE ===" > "$SCRATCHPAD/results.txt"
for slug in "${GH_SLUGS[@]}"; do
  r=$(curl -s --max-time 8 "https://boards-api.greenhouse.io/v1/boards/${slug}/jobs")
  if echo "$r" | grep -qi '"jobs"'; then
    echo "$r" | python3 -c "
import sys,json,re
try:
    for j in json.load(sys.stdin).get('jobs',[]):
        t=j.get('title','')
        if re.search(r'intern|co.?op',t,re.I):
            print(f'[$slug] {t} | {j.get(\"location\",{}).get(\"name\",\"\")} | {j.get(\"absolute_url\",\"\")}')
except: pass
" >> "$SCRATCHPAD/results.txt" 2>/dev/null
  fi
done

LEVER_SLUGS=(palantir hermeus anduril shieldai skydio jobyaviation archeraviation
  stokespace relativity dedrone fortem kratos betatechnologies supernal ursamajor
  venusaero firehawk sarcos xwing zipline trueanomaly apexspace vardaspace
  starfishspace turionspace wiskaero)

echo "=== LEVER ===" >> "$SCRATCHPAD/results.txt"
for slug in "${LEVER_SLUGS[@]}"; do
  r=$(curl -s --max-time 8 "https://api.lever.co/v0/postings/${slug}?mode=json")
  if echo "$r" | python3 -c "import sys,json; d=json.load(sys.stdin); exit(0 if isinstance(d,list) else 1)" 2>/dev/null; then
    echo "$r" | python3 -c "
import sys,json,re
try:
    for j in json.load(sys.stdin):
        t=j.get('text','')
        if re.search(r'intern|co.?op',t,re.I):
            print(f'[$slug] {t} | {j.get(\"categories\",{}).get(\"location\",\"\")} | {j.get(\"hostedUrl\",\"\")}')
except: pass
" >> "$SCRATCHPAD/results.txt" 2>/dev/null
  fi
done
cat "$SCRATCHPAD/results.txt"
```

**Step 2 — Single web search agent:**

Paste the contents of $SCRATCHPAD/results.txt, then ask one agent to:
- Search for 2027 intern roles at companies not found in Step 1 (Anduril, Shield AI, Skydio, Joby, Archer, Wisk, Beta, companies in Manual Check section)
- Search Greenhouse/Lever for Bucket B startups: `site:greenhouse.io "summer 2027" intern aerospace`, `site:lever.co "summer 2027" intern defense`
- Return company | role | location | URL | term for every new finding

**Step 3 — Verify existing links:**

For Greenhouse job URLs: `curl -s -o /dev/null -w "%{http_code}" <url>` — 302 redirect to `?error=true` means closed.
For Lever posting URLs: check via `https://api.lever.co/v0/postings/<company>/<id>` — 404 means closed.

**Step 4 — Overwrite README.md with verified results using existing section structure.**

Then: `git add README.md && git commit -m "chore: update tracker — $(date +%Y-%m-%d)" && git push`
```
